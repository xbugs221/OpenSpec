## REMOVED Requirements

### Requirement: Update Behavior
**Reason**: Runtime-only OpenSpec no longer refreshes prompts, skills, or managed agent files.
**Migration**: Update external prompts and skills in their owning repositories or home directories.

### Requirement: Prerequisites
**Reason**: The update command is retired.
**Migration**: None.

### Requirement: File Handling
**Reason**: OpenSpec no longer rewrites managed AI tool files.
**Migration**: Manage those files directly.

### Requirement: Tool-Agnostic Updates
**Reason**: Tool-agnostic prompt refresh is no longer part of the product.
**Migration**: None.

### Requirement: Core Files Always Updated
**Reason**: Runtime-only OpenSpec no longer owns AGENTS/prompts refresh flows.
**Migration**: Keep project guidance in version-controlled docs outside OpenSpec update automation.

### Requirement: Slash Command Updates
**Reason**: Slash command generation and refresh are removed with the distribution pipeline.
**Migration**: Maintain slash commands manually if still needed.

### Requirement: Archive Command Argument Support
**Reason**: Archive prompt templates are no longer generated by OpenSpec.
**Migration**: Implement archive argument handling in your external prompt definitions.

### Requirement: Error Handling
**Reason**: Update-time file mutation is removed.
**Migration**: None.
